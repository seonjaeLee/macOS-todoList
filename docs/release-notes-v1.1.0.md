# v1.1.0 릴리스 노트 (GitHub Release · 공개 레포용 초안)

## 요약

**항상 위에 띄우기** 기능 추가, 사용 가이드·타이틀바 UI(더보기 메뉴) 정리, macOS에서도 보이는 커스텀 툴팁(panel) 적용.

## 추가 · 변경 (v1.0.0 → v1.1.0)

- **항상 위에 띄우기** (`alwaysOnTop`): 메모 타이틀바 열기구(넓은 창) 또는 더보기 메뉴·메모 목록에서 다른 창·앱 위에 표시. 드래그·리사이즈 가능, 재시작 후에도 유지.
- **메모 목록 연동**: 목록에서도 열기구로 on/off. 띄워 둔 중 파란색 표시, 메모 창과 실시간 동기화.
- **타이틀바 UI**: 좁은 창은 `⋯` 더보기 + `✕` 닫기. 넓은 창은 열기구·`＋`·색상환을 타이틀바에 펼침.
- **사용 가이드**: 메모 관리(목록·모두 숨기기/보이기·닫기), 항상 위에 띄우기(메모·목록), 열기구 아이콘 안내.
- **타이틀바 툴팁**(메모 창, panel): `메모지 색상 변경` · `메모추가` · `항상 위에 띄우기` · `더보기` · `닫기`
- **툴팁 표시 보완**: frameless 창에서 HTML `title` 미표시 → 메모 창은 `data-tooltip` + `tooltip.html` panel. 메모 목록은 CSS `[data-tooltip]::before` 유지.

## ⚠️ Apple Silicon / Intel Mac

- **Apple Silicon (M1/M2/M3 등)**: `todoList-myfunfun-1.1.0-arm64.dmg` 사용
- **Intel Mac**: arm64 빌드와 호환되지 않을 수 있음. Intel용 빌드가 별도 제공될 때까지 Apple Silicon Mac에서의 사용을 권장

## 설치

1. 아래 **Assets**에서 `.dmg` 다운로드
2. dmg를 열고 앱을 **응용 프로그램** 폴더로 드래그
3. 기존 `todoList-myfunfun`이 있으면 교체 후 실행
