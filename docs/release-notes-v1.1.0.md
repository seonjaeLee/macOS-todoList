## 추가 · 변경 (v1.0.0 → v1.1.0)

### 새 기능

- **항상 위에 띄우기** (`alwaysOnTop`): 메모 타이틀바 열기구(넓은 창) 또는 더보기 메뉴·**메모 목록**에서 다른 창·앱 위에 표시. 드래그·크기 조절 가능, 재시작 후에도 유지.
- **메모 목록 연동**: 목록에서 열기구로 on/off. 띄워 둔 중 파란색 표시, 메모 창과 실시간 동기화.
- **타이틀바 UI**: 좁은 창은 `⋯` 더보기 + `✕` 닫기. 넓은 창은 열기구·`＋`·색상환을 타이틀바에 표시.
- **사용 가이드**: 메모 관리(목록·모두 숨기기/보이기·닫기), 항상 위에 띄우기, 더보기 메뉴, 빈 화면 우클릭, 열기구 아이콘 안내.
- **툴팁**(macOS panel): 메모 타이틀바·메모 목록 아이콘에 표시 (`메모지 색상 변경` · `메모추가` · `항상 위에 띄우기` · `더보기` · `삭제` 등). 닫기 `✕` 툴팁 없음.
- **더보기(⋯) 메뉴**: 빈 바닥 우클릭 메뉴·툴팁과 같은 다크 스타일, 너비·글자 크기 동일.

### 개선

- **메모 세로 정렬**: 같은 열(가로 위치가 비슷한 메모)끼리 접기/펼치기 시 **1px 간격**으로 다시 쌓임. 앱 시작 시에도 간격 복구.
- **개요 입력란**: 메모를 펼치면 저장된 내용만큼 높이가 늘어남(최대 **140px**, 넘치면 스크롤).
- **편집 단축키**: 메모 **제목**·**할 일 추가** 입력창에서 **전체 선택·복사·붙여넣기**(`⌘A` / `⌘C` / `⌘V`). 메뉴바 **편집** 메뉴 지원.

### 버그 수정

- 메모를 접었다 펼쳤을 때 **아래 메모 간격이 비어 남던** 문제.
- **왼쪽·오른쪽 두 열**이 한꺼번에 재정렬되며 **중간에 빈 간격**이 생기던 문제.
- 메모를 **숨긴 뒤**(✕ 닫기) 보이지 않는 메모 때문에 **빈 칸**이 남던 문제.
- 펼친 메모의 **개요**가 한 줄만 보이던 문제(접힌 상태에서 높이를 잘못 잡던 경우).

## ⚠️ Apple Silicon / Intel Mac

- **Apple Silicon (M1/M2/M3 등)**: `todoList-myfunfun-1.1.0-arm64.dmg` 사용
- **Intel Mac**: arm64 빌드와 호환되지 않을 수 있음. Intel용 빌드가 별도 제공될 때까지 Apple Silicon Mac 사용을 권장합니다.

## 설치

1. GitHub **Releases** → **v1.1.0** → **Assets**에서 `.dmg` 다운로드
2. dmg를 열고 앱을 **응용 프로그램(Applications)** 폴더로 드래그
3. 기존 `todoList-myfunfun`이 있으면 교체 후 실행

### 실행이 막힐 때 (macOS)

- `"손상되어 열 수 없습니다"`: 앱 손상이 아닌 미서명 앱 안내입니다. **우클릭 → 열기** 또는 **시스템 설정 → 개인 정보 보호 및 보안 → 그래도 열기**
- 터미널: `xattr -cr /Applications/todoList-myfunfun.app`
