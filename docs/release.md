# 배포 절차 체크리스트

## 1) 개발 레포 준비

- 기능 개발/수정 완료
- 로컬 실행 및 기본 동작 확인
- 변경 이력 `history.md` 기록

## 2) 빌드

- `npm run build`
- 생성된 앱 번들 점검

## 3) 공개 레포 반영

- 공개 레포 `README.md` 설치 가이드 최신화
- 릴리스 노트 작성
- 설치 파일(zip/dmg) 업로드 준비

## 4) Release 게시

- 태그 생성(`vMAJOR.MINOR.PATCH`)
- GitHub Release 생성
- 설치 파일 첨부 후 게시

## 5) 배포 후 확인

- 설치 링크 동작 확인
- 새 사용자 기준 설치 테스트
- 이슈 채널 정상 확인
