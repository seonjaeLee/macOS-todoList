# 타이틀바 간격·아이콘 직접 조정하기

`/Applications`에 설치된 앱은 **빌드할 때 CSS가 묶여** 있어서, `widget.css`만 고쳐도 화면에 바로 안 바뀝니다.

## 타이틀바 두 가지 모드

메모 창 **가로 너비**에 따라 `body.titlebar-expanded` 클래스가 토글됩니다.

| 모드 | 조건 | 타이틀바에 보이는 것 |
|------|------|----------------------|
| 기본(좁음) | `titlebar-expanded` 없음 | `⋯` 더보기, `✕` 닫기 |
| 펼침(넓음) | `titlebar-expanded` | 열기구, `＋`, 색상환, `✕` 닫기 (`⋯` 숨김) |

메모 추가·색상·항상 위에 띄우기는 좁은 창에서는 **더보기 메뉴** 또는 메뉴 **⌘N**으로 사용합니다.  
간격·크기를 볼 때는 창을 **넓혀** 펼침 모드에서 확인하는 것이 좋습니다.

---

## 빠르게 확인하는 방법

1. 메뉴바 앱 **완전 종료** (`Cmd+Q`)
2. 프로젝트 폴더에서:

```bash
npm start
```

3. `widget.css` **맨 위** `:root` 변수만 수정:

| 변수 | 의미 | 기본값 |
|------|------|--------|
| `--tb-toolbar-gap` | 펼침 모드: 열기구·`＋`·색상환 사이 | `6px` |
| (열기구 `margin-right: -4px`) | 열기구↔`＋`만 추가로 4px 좁힘 | 고정 |
| `--tb-gap` | 툴바 묶음 ↔ `✕` 닫기 | `10px` |
| `--tb-btn` | 버튼 칸 크기 (`⋯`·닫기 등) | `22px` |
| `--tb-add-size` | `＋` 글자 크기 | `26px` |
| `--tb-float-icon` | 열기구 SVG | `15px` |
| `--tb-palette-size` | 색상환 지름 | `18px` |

4. 메모 창을 **넓힌 뒤** **`Cmd+R`** → 새로고침 후 확인

5. 만족하면:

```bash
npm run build
```

## 예시

```css
:root {
  --tb-gap: 12px;        /* 툴바 묶음 ↔ 닫기 간격 넓히기 */
  --tb-add-size: 28px;   /* ＋ 더 크게 */
}
```
